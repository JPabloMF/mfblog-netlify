# Data types

In this post, you will learn about every data type that JavaScript has.

**We have 7 data types in JavaScript:**

1. Number
2. BigInt
3. String
4. Boolean
5. null
6. undefined
7. Object
8. Symbols



## Number

The number type represents both integer and floating point numbers (decimals).

```javascript
let number = 34
let decimal = 34.00
```
